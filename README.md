This project requires a 'base' directory, containing apache ant & a JDK, on top-level, e.g.:

```
workshop/
|-- base
|   |-- apache-ant-1.10.14
|   |-- jdk-21.0.3
|-- simpleApp
|   |-- app
|   |   |-- classes
|   |   |-- jar
|   |   |-- lib
|   |   |-- src
|   |       `-- de
|   |           `-- jk
|   |-- ant_do_stuff.sh
|   `-- buildfile.xml
`-- README.md
```
